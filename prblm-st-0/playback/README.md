Space to Ellipsis Converter
Description
This Python script takes user input and replaces every space " " with an ellipsis "...". It’s a quick example of using Python’s .replace() method for simple text transformation.

How It Works
The program prompts the user to enter a line of text.
It replaces every space character with three dots (...).
It prints the modified version of the text.
Example Usage
Input:
Hello world this is cool

Output:
Hello...world...this...is...cool
