# Text Lowercase Converter

## Description
This simple Python script takes a line of text from the user, removes any leading or trailing whitespace, converts all characters to lowercase, and prints the result.
Itâ€™s a basic example of string manipulation and input handling in Python.

---

## How It Works
1. The program prompts the user for input.
2. It removes extra spaces at the beginning and end using `.strip()`.
3. It converts all letters to lowercase using `.lower()`.
4. Finally, it prints the cleaned, lowercase version of the text.

---

## Example Usage

### Input:
Hello THERE, Caleb!

### Output:
hello there, caleb!
